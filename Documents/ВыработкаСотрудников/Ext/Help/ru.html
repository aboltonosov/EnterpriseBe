Выработка сотрудников

Документ предназначен для регистрации факта выполнения работ сотрудниками и передачи сведений для начисления заработной платы (если включен расчет зарплаты).

Для возможности расчета зарплаты требуется включение функциональной опции Использовать программу для кадрового учета и расчета заработной платы в разделе НСИ и администрирование - Кадры - Настройки кадрового учета и штатного расписания.

Документ регистрирует в подсистеме по расчету заработной платы данные по показателю Сдельный заработок.

Операция

Характер оформляемых работ зависит от значения поля Операция:

  • Производство по распоряжениям - для работ по Маршрутным листам производства, Этапам производства, Производству продукции без заказов 2.2. При проведении документа проверяется наличие остатков трудозатрат по распоряжениям, осуществляется их списание.
  • Производство без распоряжений - для работ по Производству продукции без заказов 2.1. В документе указывается подразделение, за которым закрепляются выполненные работы, и перечень видов работ. Распределение трудозатрат на выпуски продукции без заказов выполняется документом Списание затрат на выпуск. Если в производственном подразделении документ Списание затрат на выпуск не используется, то регистрация трудозатрат недоступна.
  • Ремонт - для работ по выполнению Заказов на ремонт.
  • Прочие работы - для прочих работ.

Виды работ
  • Для операции Производство в табличной части Виды работ указывается Группа (вид) продукции, для производства которой выполнялись работы. При заполнении документа по распоряжениям Группа (вид) продукции заполняется автоматически и не доступна для изменения. Расходы, связанные с выполнением таких работ, будут отнесены на партии продукции, указанные в распоряжениях.
  • Для операции Ремонт в табличной части Виды работ указываются Заказы на ремонт, Способ отражения зарплаты. Расходы, связанные с оплатой выполненных работ, будут отнесены на статьи расходов, указанных в способе отражения.
  • Для операции Прочие работы в табличной части Виды работ указываются Способы отражения зарплаты. Расходы, связанные с оплатой выполненных работ, будут отнесены на статьи расходов, указанных в способе отражения.

Сотрудники

Табличная часть Сотрудники содержит сотрудников организации и может быть заполнена автоматически (кнопка Заполнить) по текущему составу бригады.

Если переключатель Применять распределение по КТУ установлен, то распределение между сотрудниками выполняется пропорционально коэффициентам трудового участия для документа в целом, в противном случае для каждой строки табличной части Виды работ указываются исполнители и объем выполненных работ.
