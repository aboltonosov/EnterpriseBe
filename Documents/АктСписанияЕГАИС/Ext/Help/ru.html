Акт списания ЕГАИС

Документ предназначен для выгрузки в ЕГАИС информации о списываемой алкогольной продукции.

Документ позволяет списывать продукцию из регистра №1 или из регистра №2. Для этого предназначен реквизит Списать из. При списании из регистра №1 требуется обязательно указать справку 2, по которой товар был принят.

В реквизите Причина списания требуется выбрать причину, на основании которой формируется акт. Причины списания Иные цели и Реализация доступны только при списании из регистра №2 или при использовании формата обмена V2.

Документ в статусе Передан в ЕГАИС формирует расход по регистру накопления Остатки алкогольной продукции ЕГАИС.
