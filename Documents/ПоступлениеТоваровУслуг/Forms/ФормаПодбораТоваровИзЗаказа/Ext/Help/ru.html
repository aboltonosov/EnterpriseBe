Форма подбора товаров из заказа

В форме находится список товаров из распоряжения, на основании которого оформлен документ, а также товаров из складских ордеров, оформленным по этим распоряжениям. В форме необходимо флажками отметить те позиции из распоряжения, которые необходимо перезаполнить в документе. Если в распоряжении были добавлены новые строки, то такие строки при открытии формы по умолчанию будут отмечены флажками. По команде Перенести в документ информация по отмеченным строкам будет в документе обновлена автоматически.
Также можно заполнить накладную всем количеством из ордеров или из заказов специальной командой.
