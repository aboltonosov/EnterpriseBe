Отчеты давальцам

Форма Отчеты давальцам предназначена для оформления услуг по переработке давальческого сырья. Форма состоит из двух закладок:

  • Отчеты давальцам - список оформленных документов с отбором по менеджеру по переработке давальческого сырья.
  • Распоряжения на оформление - это список заказов давальца, по которым оформлен выпуск продукции и отчеты еще не сформированы.

Создание отчета давальцу доступно как по команде Создать так и по распоряжению.

Оформление по распоряжению

Оформление по распоряжению

Для оформления отчета давальцу по распоряжению требуется перейти на закладку Распоряжения на оформление, выделить одно или несколько распоряжений, с совпадающими основными реквизитами Организация, Давалец, Договор и нажать команду Оформить отчет давальцу. Формирование одного отчета по нескольким заказам регулируется функциональной опцией Отчет и передача давальцу по нескольким заказам в разделе НСИ и администрирование - Продажи - Производство из давальческого сырья.

В открывшейся форме нового отчета давальца основные реквизиты и табличная часть будет заполнена по данным распоряжений.
