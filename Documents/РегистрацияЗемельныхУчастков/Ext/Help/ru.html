Регистрация земельных участков

Документ предназначен для регистрации перечня объектов эксплуатации и основных средств для целей налогообложения по земельному налогу.

Реквизиты закладки Налоговый орган используются при необходимости уточнения постановки объекта на учет в налоговом органе, отличающемся от указанного для организации по умолчанию.

На закладке Земельные участки перечисляются регистрируемые для целей налогообложения по земельному налогу объекты. В качестве которых могут быть перечислены: принятые к учету основные средства, готовящиеся ко вводу в эксплуатацию объекты основных средств, не принимаемые к учету или находящие на забалансовом учете объекты эксплуатациии. При этом, в качестве Группы ОС для перечисленных объектов, должно быть задано значение Замельные участки.

На закладке Способ отражения расходов в строках табличной части необходимо указать Подразделение и Статью и аналитику расходов, на которые будет распределяться земельный налог.
Коэффициент пропорции (может быть, как дробным, так и целым числом) определяет, какая часть будет отнесена на данные Подразделение и Статью/аналитику расходов указанные в строке.

Примечание. В случае если проводится регистрация земельных участков на дату старше 2008 г., в табличной части для каждого обязательны к указанию Дата начала проектирования.
