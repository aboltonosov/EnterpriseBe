Рабочее место

Рабочее место позволяет увидеть списки как сформированных документов, так и невыполненных распоряжений на оформление. Списки представлены на трех закладках.

Оформлено

К оформлению (Передача в подразделение)

К оформлению (Передача из подразделения)

Оформлено

На закладке отображается перечень оформленных документов движения продукции и материалов. Из этого же списка можно создать новый документ любой выбранной операции.

К оформлению (Передача в подразделение)

На закладке отображается перечень распоряжений, по которым необходимо оформить документы передачи в подразделение или цеховую кладовую. Распоряжением может быть заказ материалов в производство или этап производства по материалам, описанным в обеспечении этапа производства. Документ передачи может быть оформлен по одному или нескольким выделенным распоряжениям, если в них совпадают основные реквизиты.

Для индикации состояния оформления документов движения и расходных ордеров на материалы, оформленных по распоряжениям, предусмотрены две информационные колонки с пиктограммами. Пояснение назначения пиктограмм приведено в соответствующих колонкам отборах. По команде Оформить по распоряжениям создается документ и заполняется всеми позициями распоряжений, по которым не оформлены документы. По команде Оформить по ордерам создается документ движения и заполняется всеми отгруженными по ордерам позициями.

К оформлению (Передача из подразделения)

На закладке отображается перечень распоряжений, по которым необходимо оформить документы передачи из подразделения. Распоряжением может быть этап производства по материалам, описанным к возврату в неизрасходованных материалах (операция Возврат материалов из производства), либо по выходным изделиям (операция Передача продукции из производства).
Документ передачи из подразделения может быть оформлен по одному или нескольким выделенным распоряжениям, если в них совпадают основные реквизиты: операция, отправитель, получатель, организация.
Индикация состояния оформления документов движения и ордеров аналогична закладке К оформлению (Передача в подразделение).
