Движение продукции и материалов

Документ предназначен для отражения в системе движения продукции, материалов и полуфабрикатов между складами, производственными подразделениями и цеховыми кладовыми в рамках выполнения следующих операций:

  • Передача продукции из производства
  • Передача продукции из кладовой
  • Передача материалов в производство
  • Передача материалов в кладовую
  • Перемещение материалов в производстве
  • Возврат материалов из производства
  • Возврат материалов из кладовой

Документ может создаваться как самостоятельно, так и по распоряжениям. В качестве распоряжений могут выступать заказы материалов в производство и этапы производства.
Основное рабочее место для работы с документом расположено в меню Производство - Материальный учет - Движение продукции и материалов.

Документ доступен, если включена опция Производство - Управление производством "Версия 2.2".

Передача продукции из производства

Используется для отражения передачи произведенной продукции из подразделения на склад или цеховую кладовую чужого подразделения. Передача осуществляется на основании этапа производства.

Передача продукции из кладовой

Используется для отражения передачи произведенной продукции из цеховой кладовой на склад. Передача осуществляется без распоряжения.

Передача материалов в производство

Операция используется для описания передачи материала со склада в подразделение. Передача осуществляется на основании этапа производства.

Передача материалов в кладовую

Операция используется для описания передачи материала со склада в цеховую кладовую. Передача осуществляется без распоряжения или по заказу материалов в производство. При передаче по распоряжению возможно использование строк сверх заказа.

Перемещение материалов в производстве

Предназначено для отражения перемещения между цеховыми кладовыми разных подразделений. Выполняется без распоряжений.

Возврат материалов из производства

Используется для возврата на склад из подразделения материалов, ранее переданных туда. Передача осуществляется на основании этапа производства в состоянии начат или завершен, у которого заполнены неизрасходованные материалы. Возврат осуществляется из подразделения, исполняющего этап.

Возврат материалов из кладовой

Используется для возврата на склад из цеховой кладовой материалов, ранее переданных туда. Возврат осуществляется без распоряжения.
