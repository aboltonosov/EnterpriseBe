Заказ на производство

Заказ на производство предназначен для оформления в системе потребности к производству продукции / выполнению работ и управления исполнением этой потребности. Документ доступен при включении функциональной опции НСИ и администрирование - Производство - Управление производством (версия 2.2).
Документ может использоваться как для обеспечения внешней по отношению к производству потребности (клиентские, внутренние заказы и т.п.), так и внутрипроизводственной (потребность в полуфабрикатах).
В заказе на производство описывается партия продукции/работ, объединенные одной датой потребности и запуска, единым приоритетом и направлением планирования.

Основные реквизиты заказа

Подразделение-диспетчер

Диспетчирующее подразделение, в ответственности которого находится заказ на производство.

Желаемая дата выпуска

Дата, к которой необходимо произвести произвести всю продукцию заказа

Начать не ранее

Дата, ограничивающая начало производства по заказу

Направление размещения

Определяет способ планирования производства по заказу. Может иметь значения:

  • К началу - планирование производства вперед, запуск производства как можно раньше, начиная от даты Начать не ранее.
  • К окончанию - планирование производства назад от желаемой даты выпуска, для получения результата как можно ближе к желаемой дате.

Приоритет

Определяет очередность планирования заказа. Заполнятся значениями из справочника приоритетов. Заказы с более высоким приоритетом могут вытеснять низкоприоритетные заказы. Итоговый приоритет планирования заказа определяется на 3 уровнях:

  • Приоритет заказа
  • Приоритет подразделения-диспетчера
  • Очередь заказов.

Среди заказов с одинаковым значением реквизита Приоритет на очередность влияет приоритет подразделения-диспетчера. При одинаковом приоритете и подразделении играет роль место в очереди. Заказ становится в конец своей очереди при переводе в статус К производству.

Операция

Определяет тип хозяйственной операции, выполняемой заказом. Возможные значения:

  • Собственное производство - производство продукции с использованием только своих собственных материалов.
  • Производство из давальческого сырья - производство продукции для давальца, предоставляющего свои материалы для производства. При выборе этой операции в заказе появляются реквизиты давалец и договор.

Продукция

Табличная часть содержит список продукции, которую необходимо произвести в рамках заказа на производство, с указанием количеств.
Кроме того, указывается:

  • Назначение - источник обособленной потребности, в случае производства продукции в обеспечение обособленной потребности
  • Стандартная спецификация - спецификация, с помощью которой будут сформированы этапы производства. Спецификация может быть не указана. В таком случае этапы производства продукции необходимо будет сформировать вручную. При добавлении новой строчки продукции автоматически подставляется основная спецификация, если задана для этой продукции. Если выбранная для продукции спецификация является основной спецификацией, она отмечается в списке галочкой.
  • Получатель - склад, на который необходимо поместить произведенную продукцию, или подразделение, для которого необходимо выполнить работу

Заданные в заказе приоритет, желаемые даты начала и выпуска, направление размещения едины для всей продукции заказа.

Если для какой-то продукции необходимо задать отличающиеся параметры планирования, такую продукцию нужно оформлять в отдельном заказе.

Действия
  • Заполнить основной спецификацией - для всех выбранных строк продукции будут подставлены основные спецификации, если есть.
  • Назначить спецификацию основной - указанная в строке спецификация будет назначена основной для продукции.
  • Структура заказа - открыть форму структуры заказа
