Настройка автовыбора материала

Настройка автовыбора материала используется для описания правил, по которым подбирается материал при производстве.
Настройка используется, когда материал или его характеристику нельзя указать однозначно, но можно подобрать по свойствам продукции или ее характеристикам.

Материал указывается в свойстве характеристики продукции

Характеристика материала подбирается по свойствам продукции или ее характеристике

Материал уточняется при производстве

Материал указывается в свойстве характеристики продукции

Настройка используется, когда нужный материал указан в одном из свойств характеристики продукции.

Пример
Выходное изделие – стул. Для стула указано свойство обивка с типом позиция номенклатуры. Значение свойства - кожа. Для исходного материала указано: задается в свойстве продукции - обивка. При заполнении заказа на производство материал кожа будет заполнен автоматически. Если впоследствии для стула поменяют обивку на кож.зам. - в заказах на производство автоматически будет указан этот материал.

Характеристика материала подбирается по свойствам продукции или ее характеристике

Нужная характеристика материала будет определена, по сопоставлению значений ее свойств со свойствами продукции.
В настройке соответствия свойств можно сопоставить можно либо одинаковые свойства, либо свойства, созданные по образцу.

Пример
Материал "Обивочная ткань", имеет в характеристиках свойство "Цвет ткани".

Готовая продукция "Кресло", в котором эта ткань применяется. Причем, у кресла могут быть разные цвета сиденья и спинки.
У кресла характеристики имеют 2 свойства: "Цвет сиденья" и "Цвет спинки".

Для того, чтобы можно было сопоставить эти свойства в спецификации, надо сначала сделать свойство "Цвет ткани", как новое.
А для кресла создавать свойства уже не самостоятельные, а по образцу свойства "Цвет ткани".

У свойств, созданных по образцу, набор значений будет не собственный, а взят из свойства-образца. То есть возможные цвета спинки и сиденья будут определяться доступными цветами ткани.
Тогда их можно будет сопоставлять в спецификации в автовыборе материала.

Материал уточняется при производстве

Настройка используется, когда материал нельзя определить автоматически и его требуется выбирать вручную при непосредственном производстве.
В заказе на производство будет добавлена пустая строка материала и предложено вручную выбрать нужный материал. При этом будет доступна информация о применении материала.
