Справочник "Группы основных средств и коэффициенты переоценки"

Справочник «Группы основных средств и коэффициенты переоценки»  предназначен для хранения информации о коэффициентах переоценки основных средств в зависимости от групп переоценки и шифров объектов основных средств по Временному республиканскому классификатору основных средств, которые используются для проведения переоценки основных средств по индексному методу в документе "Переоценка основных средств".

В форме элемента справочника для конкретной группы переоценки основных средств в табличной части "Шифры по ВРКОС" заполняются интервалы шифров объектов основных средств по Временному республиканскому классификатору основных средств, для которых будут применяться коэффициенты переоценки основных средст, которые указываются в табличной части "Коэффициенты переоценки".

Справочник может быть заполнен как вручную, так и при помощи загрузки из файла KOC.dbf , который входит в комплект поставки программы и обновляется по мере необходимости.

Чтобы загрузить коэффициенты необходимо указать путь к указанному файлу в форме загрузки групп и коэффициентов переоценки основных средств, указать в форме на какое число производится переоценка. Затем загрузить группы переоценки, выбрать нужные группы, проставив флажки, и загрузить коэффициенты.

Форма загрузки групп и коэффициентов переоценки основных средств открывается из справочника "Группы основных средств и коэффициенты переоценки" по кнопке "Загрузить из файла" (кнопка видна из формы списка справочника).
