Объекты эксплуатации

Справочник предназначен для регистрации объектов хозяйственной деятельности с необходимой детализацией административной и технологической структуры. Отдельные объекты эксплуатации могут быть приняты к учету как основные средства, при этом в программе соблюдается регламентированно установленный порядок их принятия к учету и других изменений в параметрах учета.

Детализация технологической структуры объекта эксплуатации осуществляется с помощью описания структуры узлов по навигационной ссылке Узлы объекта.

Для узлов возможно указать собственные классы обслуживания и независимо фиксировать наработку. Для узлов сохраняется история изменения принадлежности к объекту эксплуатации – перенос из состава одного объекта эксплуатации в состав другого. При таком переносе сохраняется наработка, накопленная узлом в составе первоначального объекта эксплуатации.

Закладка Основные


На закладке определяются основные реквизиты и параметры учета объекта эксплуатации.

  • Поле Класс обязательно к заполнениию, при использовании подсистемы управления ремонтами. При необходимости, основная классификация уточняется Подклассом;
  • Направление деятельности - сквозной аналитический разрез при планировании и учете доходов, расходов;
  • Пасспортные характеристики не обязательны для заполнения, служат для целей детализированного описания и классификации объекта эксплуатации;
  • Параметры учета затрат Ремонт обязятельные к заполнению: Статья расходов и Способ отражения зарплаты. При выполнении ремонтов материальные затраты будут отнесены на статью расходов, а трудозатраты приняты к учету в соответствии со способом отражения;
  • В группе Эксплуатация указывается Ответственное подразделение и Расположение - строка в произвольном формате для упрощения поиска объекта;
  • Реквизиты группы Регламентированный учет не обязательны к заполнению, если на момент заполнения не известны параметры классификации объекта для регламентированного учета они могут быть указаны непосредственно при Принятии к учету ОС;
  • Для целей классификации в рамках Международного учета требуется указание Группы ОС.

Закладка Параметры учета наработок


Закладка предназначена для заполнения свойств показателей наработок данного объекта эксплуатации. В случае, если заданы соответствующие настройки для перечня контролируемых показателей на этапе классификации, для каждого показателя наработки объекта можно задать Источник получения текущих значений наработки и Назначенный ресурс.

Закладка Рабочие центры


На закладке уточняется список зависимых рабочих центров, информация об ограничении доступности которых, будет предоставлена для анализа при проведении ремонтных работ на объекте эксплуатации.


Закладка Драгоценные материалы


На закладке предусмотрена возможность перечисления драгоценных материалов, входящих в состав основного средства. Доступ к этому разделу регулируется функциональной опцией Драгоценные металлы и камни в разделе НСИ и администрирование - Номенклатура - Дополнительная информация.
