Основные настройки

Для каждого вида плана указываются следующие реквизиты:

  • Тип плана использования - определяет какой план (продаж, закупок и т.п.) можно создавать по этой настройке;
  • Вариант заполнения (по формуле и источникам) - простой вариант (по формуле) используется для "тянущей" схемы планирования, когда целевым является План продаж, а остальные планы обеспечивают его исполнение, расширенный вариант (по источникам) используется для произвольного заполнения планов;
  • Вариант записи (замещающий и суммирующий) - замещающий план заменяет предыдущий, его возможно использовать для скользящего планирования, суммирующий план суммирует план с предыдущим.
  • Количество периодов по умолчанию - позволяет указать, сколько периодов должно отображаться при создании документа плана.
  • Степень детализации плана (по подразделению, назначению, партнеру, соглашению, складу, формату магазина, менеджеру). Объединенный план по каждой детализации позволяет в одном документе указывать множество различных значений. Например, составить один документ-план с указанием всех или нескольких складов;
  • Правило заполнения по умолчанию (расширенный вариант) на основании выбранных источников данных. Позволяет указывать схему соотношения различных источников при заполнении документов и операции изменения количества и цены на процент (%) и округление;
  • Ограничения на редактирование правила заполнения (расширенный вариант) - позволяет запретить редактирование правила в документе плана. Если запрет установлен, кнопка настройки правила в документе будет скрыта;
  • С указанием плана оплат - позволяет вводить планируемые поступления или списания денежных средств по планам продаж и закупок, рассчитывать этапы оплат по указанному в документе графику.

Для планов производства дополнительно задаютс следующие параметры:
Заполнять действующие спецификации:


  • На дату начала планирования - действующая спецификация для каждой строки определяется одна по дате начала планирования

  • На каждый период - действующая спецификация может меняться по периодам



Планировать полуфабрикаты - настройка задает способ планирования обеспечиваемых (не производимых в процессе) полуфабрикатов:

  • Автоматически - полуфабрикаты планируются отложено после проведения плана, одновременно с формированием потребности в материалах

  • Вручную - полуфабрикаты могут быть запланированы добавлением в план отдельными строчками или отдельными планами


Отражение в бюджетировании

Если в сценарии планирования определена возможность отражения планов в бюджетировании, то документы по Виду плана могут формировать плановый оборот по статьям бюджетов для товаров и оплат. Для этого необходимо установить соответствующие флаги в группе Отражать по статьям бюджетирования и указать статьи бюджетов, по которым отражается данный вид плана. Оплаты можно отражать только если установлен флаг Заполнять план оплат. Статьи бюджетов могут быть уточнены непосредственно в документе.
