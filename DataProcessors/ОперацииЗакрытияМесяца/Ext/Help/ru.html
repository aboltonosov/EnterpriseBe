Операции закрытия месяца

Рабочее место предназначено для последовательного выполнения регламентных операций закрытия месяца, диагностики состояния каждой операции, а также настройки автоматического выполнения операций по расписанию.

Закрытие месяца может выполняться как для указанной организации, так и одновременно для всех организаций, к которым есть доступ у пользователя, выполняющего операции закрытия.

На форме присутствуют этапы выполняемые вручную и выполняемые автоматически. Ручные этапы показываются только в случае если их необходимо выполнить, в противном случае - они не отображаются.

Выполнять ручные этапы необходимо перед выполнением автоматических. Запуск вручную автоматической операции ведет к пересчету всех невыполненных операций, располагающихся перед ней. Операции расположенные ниже при таком запуске не пересчитываются.

Процесс закрытия месяца состоит из последовательного выполнения этапов, указанных на форме. Нажатие на кнопку Выполнить операции последовательно выполняет указанные действия в автоматическом режиме. При автоматическом расчете блокируются все кнопки панели, появляется панель с выводом информации о том, какая операция рассчитывается.

Если установлен флажок Останавливаться по ошибке и если в результате расчете операции не был диагностирован статус Выполнено успешно, то работа автоматического расчета прекращается, в противном случае выполняется расчет других операций.

Для каждого этапа в текущий момент времени показывается его состояние:

  • Не требуется - состояние указывает на то, что выполнение этапа в закрываемом месяце не требуется.
  • Не выполнено - состояние указывает на то, что выполнение этапа необходимо, но в данный момент он еще не выполнялся, или предыдущая попытка его выполнения закончилась неудачей.
  • Выполнено с ошибками - состояние указывает на то, что данный этап выполнен частично.
  • Выполнено успешно - состояние указывает на то, что все действия, выполняемые в рамках этапа, выполнены успешно.

Кнопка Скрыть операции не требующие выполнения производит скрытие этапов со статусом не требуется

Дополнительные материалы на ИТС

Исправить отрицательные остатки товаров организаций

При выполнении этапа анализируется присутствие отрицательных остатков товаров на конец периода. При их наличии открывается Помощник исправления отрицательных остатков, при помощи которого предлагается исправить данные остатки.

Распределить материалы и работы на себестоимость продукции

Этап выполняется вручную. Открывается форма рабочего места документа Распределение материалов и работ, необходимо выполнить распределение остатков затрат.

Распределение расходов будущих периодов

Этап выполняется вручную. Открывается форма списка документов Распределение расходов будущих периодов, требуется создать необходимые документы распределения.

Настроить показатели для распределения расходов на себестоимость продукции

Этап выполняется вручную. Открывается форма списка справочника Показатели для распределения расходов, в которой выводятся показатели с незаполненными значениями. В данных показателях необходимо указать значения.

Списать затраты на выпуск без распоряжений

Этап выполняется вручную, открывается форма рабочего место документа Списание затрат на выпуск, необходимо списать остатки затрат.

Оформить передачи и возвраты товаров

В данном этапе контролируется необходимость оформления Передач и возвратов между организациями Если данная необходимость присутствует, то данный этап отображается на форме. При переходе по гиперссылке открывается рабочее место Оформление передач и возврат товаров между организациями

Оформить отчеты по комиссии между организациями

В данном этапе контролируется необходимость оформления Отчетов по комиссии между организациями Если данная необходимость присутствует, то данный этап отображается на форме. При переходе по гиперссылке открывается рабочее место Отчеты по комиссии между организациями

Оформить отчеты комитентам о списании

В данном этапе контролируется необходимость оформления Отчетов комитентам о списании Если данная необходимость присутствует, то данный этап отображается на форме. При переходе по гиперссылке открывается рабочее место Оформление отчетов комитенту о списании

Оформить счета-фактуры и таможенные документы

В данном этапе контролируется необходимость оформления операций по учету НДС. Если данная необходимость присутствует, то данный этап отображается на форме. При переходе по гиперссылке открывается рабочее место Помощник по учету НДС

Взаиморасчеты и переоценка валютных средств

Формирование движений по расчетам с партнерами

Этап выполняется автоматически. Производится формирование отложенных движений по регистрам расчета с партнерами для распределения авансов.

Переоценка денежных средств, кредитов, депозитов и займов

Этап выполняется автоматически. Создается документ Регламентная операция с типом операции Расчет курсовых разниц по регламентированному учету.

Формирование затрат

В данной группе производится распределение всех нематериальных затрат. Группа включает в себя следующие этапы:

Начисления по кредитам, депозитам и выданным займам

Этап выполняется вручную. Открывается форма списка документов Начисления по кредитам и депозитам, требуется создать необходимые документы начислений.

Аннулирование подарочных сертификатов

Этап выполняется автоматически. Производится создание документа Аннулирование подарочных сертификатов, который заполняется списком сертификатов к аннулированию.

Начисление оценочных обязательств (резервов) по отпускам

Операция выполняется автоматически, если в учетной политике включено формирование оценочных обязательств и расчет зарплаты выполняется в ERP. При выполнении создается документ Начисление оценочных обязательств по отпускам. Если в результате выполнения операции возникли ошибки, то наименование операции отображается в виде гиперссылки, по которой открывается список документов Начисление оценочных обязательств по отпускам. В этом случае требуется открыть документ, соответствующий организации и отчетному периоду, и заполнить вручную.

Отражение зарплаты в финансовом учете

Операция выполняется автоматически, если расчет зарплаты выполняется в ERP. При выполнении создается документ Отражение зарплаты в финансовом учете. Если в результате выполнения операции возникли ошибки, то наименование операции отображается в виде гиперссылки, по которой открывается список документов Отражение зарплаты в финансовом учете. В этом случае требуется открыть документ, соответствующий организации и отчетному периоду, и заполнить вручную.

Амортизация ОС и НМА (регламентированный учет)

Этап выполняется вручную. Открывается журнал регламентных документов, с отбором по документам Амортизация ОС и Амортизация НМА, требуется создать необходимые документы.

Амортизация ОС и НМА (международный учет)

Этап выполняется вручную. Открывается журнал регламентных документов, с отбором по документам Амортизация ОС (международный учет) и Амортизация НМА (международный учет), требуется создать необходимые документы.

Признание в налоговом учете лизинговых платежей

Этап выполняется автоматически. Создается документ Регламентная операция с типом операции Признание в НУ лизинговых платежей.

Погашение стоимости ТМЦ в эксплуатации (регламентированный учет)

Этап выполняется вручную. Открывается форма списка документа Погашение стоимости ТМЦ в эксплуатации, требуется создать необходимые документы.

Расчет налога на имущество

Этап выполняется автоматически. Создается документ Регламентная операция с типом операции Расчет налога на имущество.

Расчет транспортного налога

Этап выполняется автоматически. Создается документ Регламентная операция с типом операции Расчет транспортного налога.

Расчет земельного налога

Этап выполняется автоматически. Создается документ Регламентная операция с типом операции Расчет земельного налога.

Расчет торгового сбора

Этап выполняется автоматически. Создается документ Регламентная операция с типом операции Расчет торгового сбора.

Распределение производственных затрат

В данной группе производится распределение материальных и нематериальных затрат на выпущенную продукцию. Включает в себя следующие этапы:

Настройка распределения расходов

Этап выполняется автоматически. Если этап не удалось выполнить в автоматическом режиме, то наименование этапа отображается в виде гиперссылки, по которой открывается форма рабочего места документа Распределение расходов на себестоимость продукции, в которой необходимо распределить остатки затрат.

Расчет себестоимости

Данная группа содержит в себе этапы, которые выполняются автоматически. После успешного выполнения данных этапов появляется рассчитанная себестоимость товаров.
Группа включает в себя этапы:

Формирование движений по партиям товаров

Выполняется актуализация партионного учета и распределение доп. расходов, формируются отложенные движения по регистрам партий.

Распределение НДС

При выполнении этапа создается и проводится документ Распределение НДС

Расчет себестоимости

Создается документ Расчет себестоимости товаров с фактическим вариантом расчета. Вариант расчета (Средняя за месяц, ФИФО (взвешенная оценка), ФИФО (скользящая оценка)) берется из учетной политики.

Расчет долей списания косвенных расходов

При выполнении этапа создается документ Регламентная операция с типом операции Расчет долей списания косвенных расходов.

Распределение расходов на себестоимость продукции (автоматически)

При выполнении этапа производится распределение долей стоимостей косвенных расходов пропорционально затратам в незавершенном производстве.

Формирование финансового результата

Расчет курсовых разниц по оперативному учету

Этап выполняется автоматически. Создаются документы Расчет курсовых разниц.

Распределение доходов и расходов по направлениям деятельности

Этап выполняется автоматически. Создаются документы Распределение доходов и расходов по направлениям деятельности

Регламентированный учет

Отражение документов в регламентированном учете

Этап выполняется автоматически.
Производится формирование бухгалтерских проводок для новых (или измененных) документов и формирование записей книг покупок и продаж по приобретенным ценностям за период.
Записи в книгу покупок и продаж формируются только по тем счетам-фактурам, по которым есть задания.

Формирование проводок выполняется только при включенной опции Регламентированный учет (НСИ и администрирование - Настройка разделов - Регламентированный учет) .

Списание косвенных расходов на финансовый результат

Этап выполняется автоматически. Создается документ Регламентная операция с типом операции Списание косвенных расходов.

Формирование резервов по сомнительным долгам

Этап выполняется автоматически. Создается документ Регламентная операция с типом операции Формирование резервов по сомнительным долгам.
Этап выполняется, только если в учетной политике организации определено формирование резервов по сомнительным долгам.

Формирование финансового результата

Этап выполняется автоматически. Создается документ Регламентная операция с типом операции Формирование финансового результата.

Международный учет

Регламентные операции по закрытию месяца в международном учете
