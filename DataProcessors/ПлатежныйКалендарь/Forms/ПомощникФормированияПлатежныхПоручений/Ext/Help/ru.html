Помощник формирования платежных поручений

Платежный календарь позволяет сформировать платежные документы автоматически. Для этого используется команда Сформировать списания ДС в панели инструментов платежного календаря, которая вызывает помощник формирования платежных документов.

Контроль заявок на расходование денежных средств


При вызове команды Сформировать списания ДС Платежного календаря управление передается в форму помощника формирования платежных документов.

На закладке Контроль заявок отражаются все заявки на расходование денежных средств в статусе К оплате за период оперативного планирования и результаты контроля заявок на предмет возможности формирования платежных документов по ним.

В табличной части можно применить фильтрацию по заявкам, по которым контроль пройден или не пройден, или вывести все заявки. По умолчанию отмечаются заявки, по которым контроль выполнен успешно. По выделенным заявкам будут формироваться платежные поручения. В табличной части Результаты контроля отражены позиции, по которым заявка не удовлетворила требованиям контроля.

При выполнении контроля указания банковского счета списания денежных средств анализируется закладка Распределение по счетам заявки на расходование денежных средств.

Формирование платежных документов


На закладке Платежные документы производится формирование платежных поручений с использованием команды Сформировать списания безналичных ДС по заявкам на расходование денежных средств.

Для выгрузки платежных документов в банк применяется команда Выгрузка платежей в банк. При этом в диалоговом окне выгрузки платежных документов будут предустановлены счета организации, по которым формировались платежные поручения и платежные документы, которые были сформированы с использованием помощника.
