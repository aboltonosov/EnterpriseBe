Рабочее место предназначено для просмотра активных соглашений с клиентами, по которым производится выгрузка торговых предложений по товарам для покупателей в сервисе 1С:Бизнес-сеть.

Для возможности использования сервиса обмена торговыми предложениями требуется включение функциональной опции Обмен данными 1С:Бизнес-сеть (НСИ и администрирование - Настройка интеграции - Обмен электронными документами).
