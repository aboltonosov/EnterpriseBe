Проводки международного учета

Форма для просмотра, формирования и редактирования проводок международного учета.
Если документ проведен, а проводки по нему еще не сформированы из рабочего места Отражение документов в международном учете, то их можно сформировать по кнопке Отразить в международном учете автоматически.
Для ручного редактирования проводок нужно включить отметку Ручная корректировка проводок.
В поле комментарий будет выводится информация о причинах, по которым проводка не может быть сформирована автоматически с возможностью перехода к настройке шаблонов проводок и соответствия счетов.
